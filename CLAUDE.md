- 완전한 문장이 아닌, 개조식으로 작성

  1. 실무이론 (10분)
  2. 과제이론 (20분)
  3. 과제피드백 (20분)
  4. Q&A (10분)

## Mermaid 다이어그램 가이드라인
- mermaid 다이어그램은 다음 UML 다이어그램을 그릴 때만 사용하세요:
  - Use Case 다이어그램
  - Activity 다이어그램
  - Sequence 다이어그램
  - Collaboration 다이어그램
  - Class 다이어그램
  - Component 다이어그램
- mermaid 다이어그램은 꼭 필요할 때만 삽입하세요.
